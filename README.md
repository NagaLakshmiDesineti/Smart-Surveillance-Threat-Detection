# Smart Surveillance System for Threat Detection
An AI-powered security solution using **Computer Vision** to detect potential threats in real-time.

### Key Features:
* **Object Detection:** Uses YOLOv8 for high-speed person and weapon detection.
* **Real-time Processing:** Processes live camera feeds using OpenCV.
* **Automated Alerts:** Integrated email notification system for instant security updates.

### Tech Stack:
* Python, OpenCV, YOLOv8 (Ultralytics)
